# Asset Audio Folder

Folder ini untuk menyimpan musik latar untuk landing page.

## 📝 Instruksi

1. **Tambahkan wedding music di sini**
    - **Nama file**: `wedding-music.mp3` (PENTING: sesuai dengan yang di Blade template)
    - **Format**: MP3 recommended
    - **Duration**: 3-5 menit
    - **Bitrate**: 128-192 kbps
    - **File size**: 2-5MB

2. **Format yang didukung**
    - MP3 (recommended)
    - WAV
    - OGG
    - M4A (dalam beberapa browser)

## 💡 Rekomendasi Musik

- **Genre**: Classical, Romantic, atau Wedding Instrumental
- **Mood**: Elegant, calming, dan premium
- **BPM**: 60-90 BPM (slow tempo untuk wedding)
- **Instrument**: Piano, String Orchestra, atau Acoustic

## 🔊 Audio Quality Tips

- Jangan terlalu keras (peak normalization -3dB)
- Smooth fade-out untuk loop effect
- Tested di berbagai devices
