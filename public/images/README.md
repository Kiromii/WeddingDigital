# Public Images

Folder untuk menyimpan background images yang dapat diakses dari web.

## 📝 Instruksi

1. Letakkan background images di folder ini
2. Reference di Blade template:

```html
<div
    class="background-image"
    style="background-image: url('/images/your-image.jpg');"
></div>
```

## 💡 Tips

- Optimize images untuk web (compress dengan TinyPNG)
- Gunakan format modern (WebP jika browser support)
- JPG untuk photos, PNG untuk graphics
