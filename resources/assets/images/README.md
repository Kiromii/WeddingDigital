# Asset Images Folder

Folder ini untuk menyimpan background images untuk landing page.

## 📝 Instruksi

1. **Tambahkan image background Anda di sini**
    - Format: JPG, PNG, atau WebP
    - Ukuran: 1920x1080px atau lebih besar
    - File size: < 500KB

2. **Update image URL di Blade template**
    - Edit `resources/views/welcome-wedding.blade.php`
    - Cari line dengan `background-image: url(...)`
    - Ganti dengan path image Anda

## 💡 Rekomendasi

- Gunakan high-quality premium wedding images
- Pastikan image sudah ter-blur dengan baik (sudah di-blur di CSS)
- Gunakan tools seperti TinyPNG untuk compress
- Pilih wedding aesthetic atau elegant background
